# My Portfolio

A modern, responsive, and premium single-page portfolio website built using HTML, CSS, and JavaScript.

## ✨ Features

- Modern dark UI with neon gradients
- Fully responsive design
- Smooth scrolling animations
- Glassmorphism effects
- Interactive project cards
- Animated hero section
- Skills showcase
- Contact form section
- Social media integration

## 🚀 Tech Stack

- HTML5
- CSS3
- JavaScript
- Font Awesome

## 📂 Sections Included

- Hero Section
- About Me
- Skills
- Projects
- Experience & Certifications
- Education
- Contact

## 🎯 Purpose

This portfolio was created to showcase my skills, projects, and learning journey as a Computer Science Engineering student and aspiring Full Stack Developer.
